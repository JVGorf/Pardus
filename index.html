Pardus Faithful Theme by Lore

This theme improves the visuals while remaining true to the default 'Classic HQ' look. Point your account settings to jvgorf.github.io/Pardus/ using the https option.

Theme Edits. I.E. Things only I care about:

1. Edited the layout, nav bar, and tabs.
2. 128px[1] HQ Buildings, Planets, ships, and Mobs displayed in default.
3. 128px[1] HQ space tiles in default.
4. Enhanced planet and faction landscape visuals when landed.
5. New docking image.
6. Glow added to trade indication lights on buildings when trade is on.
7. Darkened buildings when trade is off.
8. Trade indication lights added to Player MO's.
9. Changed active trade lights on plastics buildings to yellow.
10. Notices moved down so they don't crash into the Pardus Sweetner[2] clocks.
11. Better online indicators.
12. Crispy faction icons[3].
13. Enhanced races[4].
14. Crispy mission icons.
15. 50x Crispy resource icons[5].
16. CSS edits.

[1] Default visuals on the Nav screen are set to display at 64px. This is still true but I'm using the 128px buildings, planets, ships, and mobs which get compressed to fit into a 64px area for better looking models.

[2] Ideally, the app should be edited rather than Pardus but it's not my place to do that.

[3] This is rendered in most cases. The rest, whether intentionally or mistakenly, pull images from a static source. Even though this is limited to a handful of pages, the visuals are vastly improved. I redid the 16px faction icons for the sake of the mission completion board - this is basically pixel art now but looks better.

[4] New stylized Pardus race icons no longer have that wax look. As usual, I'm using new 128px images contained at 64px. So it looks much better but with one sacrifice; the player gossip images when landed. The 128px images do not contain within the 52px divs so you only see a portion of the image. I'm not sure what happened here - they should have been set to 64px anyway. The divs are in-line with no ID so I'm not able to fix this with CSS. This combined with the fact that some race icons are pulled from a static source means a messy edit. I think Pardus developers intended for a full image pack replacement but for whatever reasons, once again, the static images are interfering with consistency.

[5] This is a work-in-progress. After improving the mission icons, the mission log, as well as other logs appear worse in contrast. The problem here is that I cannot contain larger images so I'm stuck with 16x16 px icons. Back in 2005, 1024x768 was the leading monitor size with 1366x768 gradually becoming more prevalent. In 2025, screen resolutions are higher giving these old icons a dated and blurry look. To fix this, I'm basically doing pixel art with fewer colors and harder edges.
