Pardus Faithful Theme by Lore

This theme improves the visuals while remaining true to the default 'Classic HQ' look. Point your account settings to jvgorf.github.io/Pardus/ using the https option.

Theme Details:

1. Cleaned up the layout, nav bar, and tabs.
2. Enhanced default* visuals for buildings.
3. Enhanced default* visuals for mobs.
4. Enhanced planet and faction visuals when landed on a planet.
5. Glow added to trade indication lights when trade is on.
6. Darkened buildings when trade is off.
7. Trade indication lights added to PC MO's.
8. Changed active trade lights on plastics buildings to yellow.
9. Notices moved down so they don't crash into the Pardus Sweetner** clocks.
10. Minor CSS tweaks.

* Default visuals on the Nav screen are set to display at 64px but I'm using the 128px buildings, planets, and mobs which get compressed to fit into a 64px area. The 128px buildings and mobs also replace the 96px option, but are still displayed within a 64px port.
** Ideally, the app should be edited rather than Pardus but it's not my place to do that.
