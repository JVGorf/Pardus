Pardus Faithful Theme by Lore

This theme improves the visuals while remaining true to the default 'Classic HQ' look. Point your account settings to jvgorf.github.io/Pardus/ using the https option.

Theme Edits:

1. Crispified the layout, nav bar, and tabs.
2. 128px* HQ Buildings, Planets, and Mobs displayed in default.
3. 128px* HQ space tiles in default.
4. Enhanced planet and faction landscape visuals when landed.
5. New docking image.
6. Glow added to trade indication lights on buildings when trade is on.
7. Darkened buildings when trade is off.
8. Trade indication lights added to Player MO's.
9. Changed active trade lights on plastics buildings to yellow.
10. Notices moved down so they don't crash into the Pardus Sweetner** clocks.
11. Better online indicators.
12. CSS tweaks.

* Default visuals on the Nav screen are set to display at 64px but I'm using the 128px buildings, planets, and mobs which get compressed to fit into a 64px area for a nice crisp look.
** Ideally, the app should be edited rather than Pardus but it's not my place to do that.
